## ✨ Features

- 🎨 **Fully config-based** — app name, tagline, features, pricing, links
- 🌗 **Dark, Light, or Toggle mode**
- 💜 **Custom accent color** — any hex code
- ⚡ **Hero section** — tagline, description, two CTA buttons
- 🖥 **App screenshot placeholder** — drop in your own image
- ✅ **Features grid** — 6 customizable feature cards
- 💰 **Pricing section** — Free, Pro, Enterprise with featured highlight
- 📱 **Fully responsive**
- ⚡ **Zero dependencies**

## 🚀 Quick Start

1. Clone or download the repo
2. Open `index.html` and edit the `CONFIG` block at the top
3. Deploy to GitHub Pages and share your link!
